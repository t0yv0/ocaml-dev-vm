# ocaml-dev-vm

Opinionated OCaml development VM descriptor.

* Arch Linux
* opam
* js_of_ocaml
* Emacs
* merlin
* tuareg-mode

To reconstruct the VM, use a tool called Vagrant. Go to the checkout directory and say `vagrant up`.

Work in progress.
